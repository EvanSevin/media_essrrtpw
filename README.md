# Projet Typo3 [DAWIN]

#### Starter link

    https://github.com/pitchart/dawin-typo3-cms-distribution

##### Launch a PHP server

    php -S localhost:<port> -t web/

The TYPO3 website is available at http://localhost:8080/ and the backend at http://localhost:8080/typo3/
